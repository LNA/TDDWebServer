# WebServer 
 
An HTTP server written in Java.  
 
## Requirements
  * Java "1.7.0_04"
  * Maven "3.2.1"
  * [Cob Spec]: https://github.com/8thlight/cob_spec
 
## Running the server
  Start the server with `java -jar build/libs/javaServer.jar -p 5000 -d PATH/TO/COB_SPEC/PUBLIC`
